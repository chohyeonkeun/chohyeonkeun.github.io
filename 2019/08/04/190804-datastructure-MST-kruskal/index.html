Kruskal algorithm

  1. Edge를 가중치가 작은 것에서 큰 것 순으로 정렬
  2. 트리에 Edge를 하나씩 추가
  3. 사이클이 생기면 추가하지 않는다.
  4. 최소 비용 신장 트리가 완성되면 |E| = |v|-1
